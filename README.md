# Hospital Management System

## Overview
The Hospital Management System is a web-based application designed to streamline hospital operations. It helps manage patient records, appointments, doctors, and billing in a centralized system.

## Features
- Patient registration and management
- Doctor information and scheduling
- Appointment booking and tracking
- Prescription and medical history storage
- Billing and payment management
- Database integration with SQLite

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** SQLite
- **Other Tools:** Better-SQLite3, dotenv, cors
